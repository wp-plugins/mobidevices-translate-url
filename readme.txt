Перевод ярлыков (URL) записей, рубрик и медиафайлов на английский язык

== Description ==

SEO-плагин для автоматического перевода ярлыков (URL) и названий медиафайлов с любого языка на английский, разработанный порталом <a href="http://mobidevices.ru">MobiDevices</a>.

В плагине нет никаких настроек, его достаточно всего лишь включить и он будет выполнять свое предназначение. MobiDevices Translate URL работает на основе Google Translate, что обеспечивает непревзойденное качество перевода с мгновенной скоростью. 

К тому, плагин удаляет из полученных URL предлоги. которые не влият на продвижение, однако увеличивает длину ссылок.

Немаловажным фактором является и то, что плагин MobiDevices Translate URL не "ломает" функцию single_cat_title(), правильно форматирует переведенный URL и не требует ключей API, в отличии от других плагинов с подобным функционалом.

== Installation ==

1. Установить плагин MD Translate URL
2. Активировать плагин
3. Наслаждаться автоматическим переводом URL и названий медиафайлов

== Frequently Asked Questions ==

Если плагин не работает, значит он не включен. По другому никак.
